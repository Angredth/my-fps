function placeModelOnGround(model, x, z) {

### 2. 건물 주변 지형 평탄화 방식 변경

- 기존 얇은 `BoxGeometry` 패드를 지형 위에 얹는 방식은 제거했다.
- 평탄화 영역 데이터만 먼저 등록하고, 건물 15개의 위치가 모두 결정된 뒤 실제 `floorGeo` 정점 높이를 한 번만 수정하도록 변경했다.

```js
let terrainFlattenApplied = false;
```

- 건물 footprint와 padding 내부는 완전히 평평하게 유지하고, 바깥쪽은 `smoothstep`으로 원래 지형 높이에 자연스럽게 복귀한다.

```js
function calculateFlattenedTerrainHeight(x, z) {
  const originalHeight = terrainFormula(x, z);
  ...
  return THREE.MathUtils.lerp(originalHeight, selectedArea.y, strongestWeight);
}
```

- 접합부가 더 부드럽게 보이도록 평탄화 blend 범위를 늘렸다.

```js
const BUILDING_FLATTEN_PADDING = 3.2;
const BUILDING_FLATTEN_BLEND = 10;
```

- 지형 정점 밀도를 올려 건물 주변 평탄화가 더 자연스럽게 반영되도록 했다.

### 3. 실제 지형 기준 Cannon 물리 충돌 재생성

- 기존 Cannon 지형 바디 직접 생성 코드를 `rebuildTerrainPhysicsBody()`로 분리했다.
- 평탄화 적용 전 최초 생성과 평탄화 적용 후 재생성 모두 같은 함수를 사용한다.
- 평탄화 후 기존 `terrainBody`를 제거하고 수정된 `floorGeo` 정점 기준으로 새 `CANNON.Trimesh`를 만든다.

```js
function rebuildTerrainPhysicsBody() {
  if (terrainBody) {
    physicsWorld.removeBody(terrainBody);
  }

  const position = floorGeo.attributes.position;
  ...
  terrainBody = new CANNON.Body({ mass: 0, shape: terrainShape });
  physicsWorld.addBody(terrainBody);
}
```

- 평탄화 적용 위치는 건물 생성 반복문이 끝난 직후로 고정했다.

```js
applyFlattenedGroundAreasToTerrainOnce();
```

- 평탄화 이후 월드 루팅 오브젝트도 실제 갱신된 지면 높이에 다시 맞춰 재접지한다.

### 4. 적 사격 정확도 거리 기반 오차 추가

- 적이 플레이어를 사격할 때 거리 기반 조준 오차를 적용하도록 추가했다.
- 가까운 거리에서는 비교적 정확하고, 거리가 멀어질수록 조준점이 자연스럽게 흔들린다.
- 명중 확률을 직접 조작하지 않고, 실제 조준 목표점에 오차를 넣는 방식이다.

```js
function getEnemyAimSpread(enemy, distanceToPlayer) {
  const weaponType = enemy.userData.weaponType || 'M4A1';
  ...
  return Math.min(maxSpread, baseSpread + distanceToPlayer * distanceSpreadRate);
}
```

- 무기별 정확도 설정:
  - M4A1: `baseSpread 0.3`, `distanceSpreadRate 0.025`, `maxSpread 2.2`
  - MP-153: `baseSpread 0.8`, `distanceSpreadRate 0.045`, `maxSpread 3.2`

```js
function createEnemyAimTarget(enemy, playerHead, distanceToPlayer) {
  const spread = getEnemyAimSpread(enemy, distanceToPlayer);
  ...
  return playerHead.clone().add(new THREE.Vector3(horizontalOffset, verticalOffset, depthOffset));
}
```

### 5. MP-153 중복 랜덤 오차 제거

- MP-153 적 사격에 있던 기존 `spreadHead` 랜덤 오차를 제거했다.
- MP-153은 기존처럼 3발을 발사하지만, 각 탄의 조준점은 새 거리 기반 오차 함수만 사용한다.
- M4A1과 MP-153 모두 기존 발사 쿨다운과 피해량은 유지했다.

```js
const aimTarget = createEnemyAimTarget(enemy, playerHead, distToPlayer);
applyEnemyShotToPlayer(enemyMuzzle, aimTarget, 'MP153', WEAPONS.MP153.damage);
```

```js
const aimTarget = createEnemyAimTarget(enemy, playerHead, distToPlayer);
applyEnemyShotToPlayer(enemyMuzzle, aimTarget, enemy.userData.weaponType || 'M4A1', WEAPONS.M4A1.damage);
```
