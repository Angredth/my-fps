# 라그돌 시스템 구현 완료 🎮

## 📋 구현된 기능

### 1. **물리 엔진 통합** ✅
- **라이브러리**: Cannon.js (CDN)
- **중력**: -9.82 m/s²
- **충돌**: 자동 물리 계산

### 2. **시체 라그돌 시스템** ✅
- **3-부분 구조**:
  - 머리 (질량: 1.5kg)
  - 몸통 (질량: 3kg)
  - 무기 (질량: 0.8kg)

- **연결 방식**: PointToPointConstraint
  - 머리-몸통: 자석처럼 느슨하게 연결
  - 무기-몸통: 손에서 느슨하게 떨어짐
  - 완전 분리 방지 ✓

### 3. **사망 메커니즘** ✅
- HP ≤ 0일 때 자동 활성화
- 랜덤 초기 속도 적용
- 회전력 포함
- 원래 적 모델 제거

### 4. **최적화** ✅
- **거리 기반 렌더링**:
  - 60m+: 그림자 제거
  - 35-60m: 그림자 제거
  - 35m 이내: 완전 렌더링

- **메모리 관리**:
  - 30초 이상 된 시체 자동 제거
  - 8초 이상, 80m 이상 거리: 제거
  - 최대 메모리 사용량 제한

### 5. **발소리 증폭** ✅
- **배수**: 4.5배 증폭
- **적용 범위**:
  - Tone: 0.038 → 0.171
  - Noise Burst (1): 0.026 → 0.117
  - Noise Burst (2): 0.012 → 0.054

## 🎯 사용 방법

### 게임플레이:
1. 적을 쏜다 (머리샷 = 9배 피해)
2. 적이 죽으면 → **자동으로 라그돌 활성화**
3. 시체가 물리적으로 넘어짐
4. 각 파트가 약간 떨어질 수 있음 (완전 분리 안 됨)
5. 30초 후 또는 매우 멀어지면 자동 제거

### 발소리:
- 이동할 때 더 크게 들림 (4.5배)
- WASD로 이동하면서 확인 가능

## ⚙️ 조정 가능한 파라미터

### spawnEnemy() 함수의 userData:
```javascript
hp: 440,  // 적 체력 (180 = 보통, 440 = 강함)
```

### createRagdoll()의 물체 질량:
```javascript
torsoBody: mass: 3,      // 몸통 무게
headBody: mass: 1.5,     // 머리 무게
weaponBody: mass: 0.8    // 무기 무게
```

### activateRagdoll()의 초기 속도:
```javascript
velocity: ±8 (8m/s ~ -2m/s Y축)
angularVelocity: ±5 (회전 속도)
```

### 발소리 증폭:
```javascript
amplifiedVolume = volume * 4.5  // 여기서 조정
```

## 📞 문제 해결

### 시체가 안 보임
- 라그돌이 지형 아래로 빠진 것 가능
- `createRagdoll`의 위치 Y값 확인

### 물리가 너무 느림
- `physicsWorld.step(1/60, ...)` 두 번째 인자 확인
- FPS 드롭 시 시체 자동 제거 거리 늘리기

### 발소리가 여전히 작음
- `amplifiedVolume * 4.5` 값을 더 증가 (예: 6.0)
- 브라우저 볼륨 확인

### 메모리 누수
- 자동 정리 타이머 체크: 30초/8초
- 브라우저 DevTools에서 메모리 모니터링
