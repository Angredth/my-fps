# 적 사격 정확도 거리 반영 패치

현재 적의 사격 로직에 **거리 기반 조준 오차**를 추가해 주세요.

## 목표

* 적이 가까이 있는 플레이어는 비교적 정확하게 사격
* 거리가 멀어질수록 탄이 자연스럽게 빗나감
* M4A1과 MP-153의 정확도를 별도로 설정
* 기존 발사 간격, 피해량, 시야 판정은 변경하지 않음
* 정확도는 플레이어 피격 확률을 직접 조작하지 않고 조준 지점에 오차를 추가하는 방식으로 구현

## 1. 거리 기반 조준 오차 함수 추가

적 사격 로직 주변에 다음 함수를 추가합니다.

```js
function getEnemyAimSpread(enemy, distanceToPlayer) {
  const weaponType =
    enemy.userData.weaponType || 'M4A1';

  let baseSpread;
  let distanceSpreadRate;
  let maxSpread;

  if (weaponType === 'MP153') {
    baseSpread = 0.8;
    distanceSpreadRate = 0.045;
    maxSpread = 3.2;
  } else {
    baseSpread = 0.3;
    distanceSpreadRate = 0.025;
    maxSpread = 2.2;
  }

  return Math.min(
    maxSpread,
    baseSpread +
      distanceToPlayer * distanceSpreadRate
  );
}
```

설정 의미:

* `baseSpread`: 근거리에서도 존재하는 기본 오차
* `distanceSpreadRate`: 거리 1유닛마다 증가하는 오차
* `maxSpread`: 지나치게 부정확해지는 것을 방지하는 최대 오차

## 2. 오차가 적용된 조준점 생성 함수 추가

```js
function createEnemyAimTarget(
  enemy,
  playerHead,
  distanceToPlayer
) {
  const spread = getEnemyAimSpread(
    enemy,
    distanceToPlayer
  );

  /*
   * 수평 오차는 크게,
   * 수직 오차는 조금 작게 적용한다.
   */
  const horizontalOffset =
    (Math.random() - 0.5) * spread;

  const verticalOffset =
    (Math.random() - 0.5) * spread * 0.65;

  const depthOffset =
    (Math.random() - 0.5) * spread;

  return playerHead.clone().add(
    new THREE.Vector3(
      horizontalOffset,
      verticalOffset,
      depthOffset
    )
  );
}
```

## 3. 적 사격 직전에 조준점 계산

기존 적 발사 코드에서 플레이어와 적 사이의 거리를 사용합니다.

이미 `distToPlayer` 또는 비슷한 거리 변수가 있다면 그대로 재사용하세요.

```js
const distanceToPlayer =
  enemy.position.distanceTo(
    playerObject.position
  );

const aimTarget = createEnemyAimTarget(
  enemy,
  playerHead,
  distanceToPlayer
);
```

`playerObject`은 현재 코드에서 실제 플레이어 위치를 나타내는 객체 이름으로 맞춰 주세요.

이미 계산된 `distToPlayer` 변수가 존재한다면 중복으로 다시 계산하지 마세요.

## 4. M4A1 사격 목표 변경

기존 코드가 다음처럼 플레이어 머리를 직접 조준한다면:

```js
applyEnemyShotToPlayer(
  enemyMuzzle,
  playerHead,
  enemy.userData.weaponType || 'M4A1',
  WEAPONS.M4A1.damage
);
```

다음처럼 변경합니다.

```js
applyEnemyShotToPlayer(
  enemyMuzzle,
  aimTarget,
  enemy.userData.weaponType || 'M4A1',
  WEAPONS.M4A1.damage
);
```

## 5. MP-153에도 동일한 거리 오차 적용

MP-153에 기존 랜덤 오차 코드가 있다면 제거하고 `createEnemyAimTarget()`의 결과를 공통으로 사용하세요.

예시:

```js
applyEnemyShotToPlayer(
  enemyMuzzle,
  aimTarget,
  'MP153',
  WEAPONS.MP153.damage
);
```

기존 MP-153 전용 랜덤 오차와 새 거리 오차를 동시에 적용하면 오차가 중복되므로 기존 랜덤 오차는 삭제하세요.

## 권장 정확도 설정

### M4A1

```js
baseSpread = 0.3;
distanceSpreadRate = 0.025;
maxSpread = 2.2;
```

예상 오차:

* 거리 5: 약 `0.43`
* 거리 20: 약 `0.8`
* 거리 40: 약 `1.3`
* 거리 80 이상: 최대 `2.2`

### MP-153

```js
baseSpread = 0.8;
distanceSpreadRate = 0.045;
maxSpread = 3.2;
```

예상 오차:

* 거리 5: 약 `1.03`
* 거리 20: 약 `1.7`
* 거리 40: 약 `2.6`
* 거리 55 이상: 최대 `3.2`

## 선택 사항: 이동 중인 적 정확도 감소

적이 이동 중이라면 추가 오차를 적용할 수 있습니다.

```js
function getEnemyMovementSpread(enemy) {
  const velocity =
    enemy.userData.velocity ||
    enemy.userData.moveVelocity;

  if (!velocity) return 0;

  const speed =
    typeof velocity.length === 'function'
      ? velocity.length()
      : 0;

  return Math.min(speed * 0.12, 0.8);
}
```

그리고 최종 오차 계산에 추가합니다.

```js
const movementSpread =
  getEnemyMovementSpread(enemy);

return Math.min(
  maxSpread,
  baseSpread +
    distanceToPlayer * distanceSpreadRate +
    movementSpread
);
```

현재 코드에 확실한 적 이동 속도 변수가 없다면 이 기능은 추가하지 말고 거리 기반 정확도만 구현하세요.

## 금지 사항

* 명중 확률을 `Math.random()`으로 직접 결정하지 마세요.
* 빗나간 탄을 강제로 플레이어에게 맞히지 마세요.
* 플레이어가 가까이 있어도 무조건 명중하게 만들지 마세요.
* 거리 계산을 프레임마다 여러 번 중복하지 마세요.
* 기존 발사 쿨다운과 피해량을 정확도 값으로 사용하지 마세요.
* M4A1과 MP-153의 기존 발사 속도는 변경하지 마세요.
* 적 AI, 이동, 시야 판정 등 다른 시스템은 수정하지 마세요.

## 완료 조건

1. 적과 플레이어의 거리가 멀어질수록 조준 오차가 증가한다.
2. 근거리에서는 적이 비교적 정확하게 사격한다.
3. M4A1이 MP-153보다 정확하다.
4. 오차는 최대값을 넘지 않는다.
5. MP-153의 기존 랜덤 오차가 중복 적용되지 않는다.
6. 기존 피해량과 발사 간격은 유지된다.
7. 변경된 전체 HTML 파일을 제공한다.

수정 완료 후 추가한 함수와 기존 사격 코드에서 변경한 부분을 간단히 설명해 주세요.
